# BuyerForeSight Backend Assignment

## Project Overview
This project is a User Management REST API built using Node.js and Express.

## Features
- Get all users
- Get user by ID
- Create user
- Update user
- Delete user
- Search and sorting support

## Tech Stack
- Node.js
- Express.js
- JavaScript

## API Endpoints

GET /users  
GET /users/:id  
POST /users  
PUT /users/:id  
DELETE /users/:id  

## How to Run
1. Clone repository
2. Run `npm install`
3. Run `node index.js`
4. Open http://localhost:3000
